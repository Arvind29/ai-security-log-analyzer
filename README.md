
#  AI Security Log Analyzer

Analyzes suspicious security logs using Google Gemini AI.
Identifies threat type, severity, and recommended action.

##  Tech Used
- Python
- Google Gemini API
- Cybersecurity concepts (MITRE ATT&CK, Threat Hunting)

##  How to Run

1. Install library:
pip install google-genai

2. Add your Gemini API key in the code

3. Run:
python log_analyzer.py

##  Output

LOG: Failed login attempt from IP 192.168.1.100 - 50 times
Threat Type : Brute Force Attack
Severity    : High
Action      : Block IP, enable MFA
